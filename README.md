# Flash Replay Presentation Kit

This repository hosts only public presentation material for Flash Replay marketplace pages.

It contains:

- `index.html` - GitHub Pages preview landing page.
- `modrinth-description.md` - paste-ready Modrinth Markdown.
- `curseforge-description.md` - CurseForge rich-editor copy and asset order.
- `assets/banner/` - hero and feature strip PNGs.
- `assets/gif/` - marketplace GIFs.
- `assets/video/` - compressed MP4 preview clips.
- `assets/thumb/` - video posters/title cards.
- `assets/mockup/` - presentation UI preview images.

It does not contain mod source code, Gradle files, jars, logs, private wiki notes, or build outputs.
