## Install

1. Install **NeoForge 21.1.233+** for **Minecraft 1.21.1**.
2. Download **Flash Replay** from Modrinth or CurseForge.
3. Drop the `.jar` into your `mods` folder.
4. Launch the game.
5. Open Flash Replay from the pause menu, or use Save Last when something worth saving happens.
